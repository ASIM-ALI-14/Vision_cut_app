VisionCut – AI Video Background Remover Android App

Introducing VisionCut, a fully production-ready Android application that uses on-device AI to remove the background from any video in real time — no internet connection required, no server uploads, no green screen needed. Built with Kotlin, Jetpack Compose, and Google ML Kit Selfie Segmentation, VisionCut delivers clean, professional background removal directly on the device. The app includes a built-in video trimmer, multiple background modes (green screen or transparent WebM), a processed video library, and a complete freemium monetization system using RevenueCat subscriptions and AdMob ads. Whether you're building a content creation tool, a video editing suite, or a social media app, VisionCut is the perfect ready-to-ship foundation.

Key Features

  • On-device AI background removal using Google ML Kit Selfie Segmentation
  • Morphological edge cleaning + Gaussian feathering for natural-looking results
  • Built-in video trimmer with draggable handles and frame preview
  • Green screen export (MP4) — ready for chroma-key in any video editor
  • Transparent export (WebM with real alpha channel) via FFmpeg-kit
  • Processed video library — browse, play, share, and delete your exports
  • Full-screen in-app video player with gradient overlay controls
  • Gallery save using MediaStore API (Android 10+ and legacy support)
  • Safe video sharing via FileProvider
  • Premium subscription paywall via RevenueCat with purchase and restore
  • AdMob Rewarded Interstitial ad (processing gate for free users)
  • AdMob Rewarded ad (export and share gate for free users)
  • No-internet dialog with premium upsell for offline users
  • Full dark mode support via Material 3 theme system
  • Clean architecture — ViewModel + StateFlow + Coroutines throughout
  • Fully documented, production-ready source code

Technical Stack

UI Technologies

  • Kotlin (100% — no Java)
  • Jetpack Compose (full declarative UI)
  • Material 3 design system with custom theme
  • Navigation Compose for multi-screen navigation
  • ExoPlayer (Media3) for in-app video playback
  • Jetpack Lifecycle + ViewModel

AI & Video Processing

  • Google ML Kit Selfie Segmentation (on-device, no API key needed)
  • MediaCodec for hardware-accelerated video decoding and encoding
  • OpenGL ES 2.0 + EGL for GPU-based frame encoding
  • MediaMuxer for audio + video muxing
  • FFmpeg-kit for transparent WebM export (optional)
  • MediaMetadataRetriever + MediaExtractor for video metadata

Monetization & Backend

  • RevenueCat SDK for subscription management and paywall UI
  • Google AdMob — Rewarded Interstitial + Rewarded ad formats
  • MediaStore API for gallery save (Android 10+ and legacy)
  • FileProvider for safe cross-app video sharing

App Requirements

  • Android Studio Hedgehog (2023.1.1) or later
  • Android SDK minimum API 26 (Android 8.0 Oreo)
  • Target SDK API 34 (Android 14)
  • Kotlin 1.9+
  • JDK 17
  • Google AdMob account (for ad unit IDs)
  • RevenueCat account (free tier is sufficient to start)
  • Google Play Developer account (for publishing)
